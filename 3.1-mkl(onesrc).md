## cmake(without test) && make -j install

- test need [netlib package](https://github.com/Reference-LAPACK/lapack-release)

````cmake
trash-empty && trash -rf ./* && cmake \
-DBUILD_SHARED_LIBS=ON \
-DBUILD_INDEX64=ON \
-DCBLAS=ON \
-DLAPACKE=ON \
-DBUILD_TESTING=ON \
-DCMAKE_INSTALL_PREFIX=/home/ljl/Apps/lapack \
..
````
